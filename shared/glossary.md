# Shared Glossary

| Term | Definition |
| --- | --- |
| Artifact | A durable document or record produced by a stage. |
| Stage Gate | Human decision point that controls stage transitions. |
| Handoff | Explicit transfer of responsibility between roles. |
| BRD | Business Requirements Document. |
| ADR | Architecture Decision Record. |
| NFR | Non-Functional Requirement. |
| DoD | Definition of Done for artifacts and delivery items. |
| Blocker | Dependency or issue preventing progress. |

Add domain-specific terms during analysis and keep definitions unambiguous.
